# Simple To-Do List App (Python + Tkinter)

A simple and clean To-Do List desktop application built using **Python** and **Tkinter**.  
It allows users to add tasks, delete them, mark them as completed, and saves everything locally.

---

##  Features

### ✔ Add Tasks  
Type a task and click **Add** to insert it into the list.

### ✔ Delete Tasks  
Select a task and click **Delete**.

### ✔ Mark as Completed  
Double-click a task to toggle **completed / not completed**.

### ✔ Local Storage  
Tasks are automatically saved in a local JSON file (`tasks.json`).

### ✔ Load Tasks on Startup  
When the app opens, all saved tasks are restored.

### ✔ Clear All Tasks  
One click deletes the entire list (optional button).

---

## Technologies Used

- **Python 3**
- **Tkinter** (GUI)
- **JSON** (local storage)
